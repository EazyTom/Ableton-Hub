# Quick Reference

A quick cheat sheet for common Ableton Hub tasks and shortcuts.

## Common Tasks

### Adding Projects

1. **Add Location**: File ‚Üí Add Location (or `Ctrl+L` / `Cmd+L`)
2. **Scan**: Click "Scan" button in toolbar (or File ‚Üí Scan All Locations)
3. **Wait**: Projects appear automatically when scan completes

### Opening Projects

- **Double-click** any project card to open in Ableton Live
- **Right-click** ‚Üí "Open With" to choose a specific Live version
- **Right-click** ‚Üí "Open Project Folder" to open in Finder/Explorer

### Organizing Projects

- **Add to Collection**: Drag project into collection in sidebar
- **Add Tags**: Right-click ‚Üí "Edit Tags"
- **Mark Favorite**: Right-click ‚Üí "Make Favorite" (or click üíé icon)
- **Create Collection**: Click "New Collection" in sidebar

### Searching

- **Type in search bar** to search project names, exports, tags, plugins
- **Use tempo buttons** for quick BPM filtering
- **Use date dropdown** to filter by creation/modification date
- **Switch search mode** using dropdown next to search box

### Viewing Project Details

- **Click project card** to see details in right panel
- **Double-click** to open in Live
- **Right-click** for context menu with more options

### Playing Exports

- **Click project card** with üîä icon to play first export
- **Click again** to cycle through multiple exports
- **Click on last export** to stop playback
- **View in details panel** for full playback controls

## View Modes

- **Grid View** - Card-based layout (default)
- **List View** - Table layout with sortable columns
- **Toggle**: View menu or `Ctrl+G` / `Cmd+G`

## List View Sorting

Click column headers to sort:
- **Name** - Alphabetical (A-Z or Z-A)
- **Location** - By folder path
- **Tempo** - By BPM (low to high or high to low)
- **Modified** - By date (newest or oldest)
- **Tags** - By tag names
- **Exports** - By export status

## Common Issues & Quick Fixes

### Projects Not Showing
- Check locations are added and active
- Click "Scan All Locations"
- Check View ‚Üí Show Missing Projects

### Can't Open in Live
- Check Live installations in sidebar
- Right-click Live version ‚Üí Set as Favorite
- Or right-click project ‚Üí Open With ‚Üí Select version

### Search Not Working
- Make sure scan completed successfully
- Check status bar for errors
- Try rescanning: File ‚Üí Scan All Locations

### App Running Slow
- Large number of projects may slow down scanning
- Try scanning locations one at a time
- Check Health Dashboard for project statistics

## Tips

1. **Use collections** - Much faster than searching through hundreds of projects
2. **Tag consistently** - Makes filtering easier later
3. **Set favorite Live version** - Speeds up opening projects
4. **Regular scans** - App watches for changes, but manual scans ensure everything is up to date
5. **Use tempo filters** - Great for finding projects in a specific BPM range
6. **Check Health Dashboard** - See overview of your project library
